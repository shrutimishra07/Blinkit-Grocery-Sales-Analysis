# Blinkit-Grocery-Sales-Analysis
An end-to-end Power BI project analyzing Blinkit grocery sales data to uncover actionable insights on sales performance, customer preferences, outlet efficiency, and product trends using interactive dashboards and DAX.

## Tools Used
- Power BI
- Excel/CSV
- DAX
- Data Visualization
  
- ## Key Insights
- Identified top-performing outlets.
- Analyzed sales across product categories.
- Compared outlet performance by size and location.
- Evaluated customer preferences.
- Created interactive filters for deeper analysis.
